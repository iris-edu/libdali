Check if socket action would have blocked.

Returns

◆ dlp_setsocktimeo()

int dlp_setsocktimeo ( SOCKET  socket,
int  timeout 
)

Set socket I/O timeout.

Set socket I/O timeout if such an option exists. On WIN and other platforms where SO_RCVTIMEO and SO_SNDTIMEO are defined this sets the SO_RCVTIMEO and SO_SNDTIMEO socket options using setsockopt() to the timeout value (specified in seconds).

Solaris does not implelement socket-level timeout options.

Parameters
socketNetwork socket descriptor
timeoutAlarm timeout in seconds
Returns
-1 on error, 0 when not possible and 1 on success.

◆ dlp_setioalarm()

int dlp_setioalarm ( int  timeout)

Set a network I/O real time alarm.

Set a network I/O alarm timer, the timeout is specified in seconds. The timer is disabled by setting the timeout to zero. On most platforms this will cause a SIGALARM signal to be sent to the calling process after timeout seconds have elapsed. This function does nothing under WIN.

Parameters
timeoutAlarm timeout in seconds
Returns
-1 on error and 0 on success.
